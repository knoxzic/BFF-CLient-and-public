# Best Face Forward - Public & Client Portal

A modern, Firebase-integrated public website and client portal for Best Face Forward tax prep and bookkeeping services.

## 🚀 Quick Access

**📌 [Live Preview](https://knoxzic.github.io/BFF-CLient-and-public/)**

## Features

- 🌐 **Public Website** - Marketing site with pricing and intake form
- 🔐 **Client Portal** - Secure dashboard for authenticated users
- 🔥 **Firebase Integration** - Real-time data sync and authentication
- 💳 **Pricing Plans** - Basic, Professional, and Enterprise tiers
- 📋 **Intake Forms** - Collect client information with Firebase storage
- 🎨 **Admin View Styling** - Dynamic dark theme for admin users
- 📱 **Responsive Design** - Beautiful UI with Tailwind CSS

## Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/knoxzic/BFF-CLient-and-public.git
   ```

2. **Open locally**
   - Open `index.html` in your browser, or
   - Run a local server: `python3 -m http.server 8000`

3. **View online**
   - Navigate to: https://knoxzic.github.io/BFF-CLient-and-public/

## Login Features

- **Admin Login**: Use credentials `admin` / `password123` for full admin portal experience
- **Quick Demo**: Click demo buttons for instant access to client portal
- **Firebase Auth**: Real Firebase authentication for production use

## Technology Stack

- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Styling**: Tailwind CSS with custom admin themes
- **Backend**: Firebase (Auth, Firestore)
- **Icons**: Font Awesome 6.5.1

## Firebase Config

Connected to `bestfaceforward-a5f69` Firebase project. Update credentials in `index.html` script section to connect your own Firebase project.
